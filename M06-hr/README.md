# M06 HR 场景：使用 OpenCode 提升 HR 效率

> **适用对象**：HR 专员、HR 助理、HR 通才  
> **所需工具**：OpenCode Desktop  
> **难度**：⭐⭐ 中等  
> **预计时间**：约 60-90 分钟

你不需要任何编程基础。只需要用自然语言告诉 OpenCode 你想做什么，它会自动编写 Python 代码来完成文件处理任务。

---

## 📖 智能办公工作原理

**OpenCode 如何帮你处理 HR 工作？**

```
你输入自然语言 → OpenCode 理解需求 → 自动编写 Python 代码 → 分析数据/处理文件 → 输出结果
```

**以本模块为例**：

| 步骤 | 你做的 | OpenCode 做的 |
|------|--------|--------------|
| 1 | 输入："请分析 HR 经理指标数据" | 理解需求，规划任务 |
| 2 | 等待 | 编写 Python 代码读取 Excel |
| 3 | 等待 | 计算关键指标、识别问题经理 |
| 4 | 等待 | 生成可视化图表 |
| 5 | 收到文件：`hr_manager_analysis.xlsx` | 汇报完成 |

**你需要知道的**：
- ✅ 不需要写代码，用自然语言描述需求即可
- ✅ OpenCode 生成的代码是透明的，你可以查看和学习
- ✅ 如果结果不满意，可以告诉 OpenCode 修改
- ✅ 生成的文件会保存在 `materials/` 目录
- ✅ **AI 的输出可能有偏差，这是正常的**——你可以像和同事交流一样不断调整

---

## 实验一：Excel HR 数据分析

### 场景描述

你是 Contoso 公司的 HR 分析师。CFO Holly Dickson 提供了一份包含经理级绩效和参与度指标的 Excel 表格。

你需要分析：
- 哪些经理的参与度低于 70%
- 哪些团队的离职率高于 15%
- 培训完成率与绩效评级的相关性
- 团队健康排名

---

### 第一步：分析经理指标数据

打开 OpenCode Desktop，在对话框中输入以下提示词：

---

#### 🟢 入门版提示词（推荐）

```
请读取 materials/manager_metrics.xlsx，
分析 HR 经理绩效数据。

要求：
1. 计算平均参与度、平均离职率、平均培训完成率
2. 找出参与度低于 70% 的经理
3. 找出离职率高于 15% 的团队
4. 提供改进建议

将分析结果保存为新的 Excel 文件，
文件名：hr_manager_analysis.xlsx
```

等待 OpenCode 处理完成（约 2-3 分钟）。

---

#### 🟡 优化版提示词（更详细的分析）

<details>
<summary>📋 点击展开优化版提示词</summary>

```
请读取 materials/manager_metrics.xlsx，
完成以下分析任务：

**分析要求**：
1. 计算关键指标：
   - 平均参与度、平均离职率、平均培训完成率
   - 各指标的分布情况
   
2. 识别问题经理：
   - 参与度 < 70% 的经理名单
   - 离职率 > 15% 的团队名单
   - 同时满足两个条件的高风险经理
   
3. 相关性分析：
   - 培训完成率与绩效评级的相关性
   - 参与度与离职率的相关性
   
4. 团队健康排名：
   - 综合参与度、离职率、培训完成率
   - 生成 Top 5 和 Bottom 5 排名

**数据列名**：
- Manager Name（经理姓名）
- Department（部门）
- Engagement Score (%)（参与度）
- Attrition Rate (%)（离职率）
- Training Completion (%)（培训完成率）
- Avg Team Tenure (Years)（平均团队任期）
- Internal Promotions (Past Year)（内部晋升数）
- Performance Rating (1-5)（绩效评级）

**输出要求**：
- 新建"分析摘要"工作表
- 包含问题经理名单
- 包含相关性分析结果
- 生成可视化图表（热力图、柱状图）

保存为：hr_manager_analysis.xlsx
```

</details>

---

### 第二步：生成可视化图表（可选）

如果需要更多图表来直观展示分析结果，继续输入：

```
请在 hr_manager_analysis.xlsx 中，
生成以下图表：

1. 经理参与度排名（柱状图，按参与度排序）
2. 离职率 vs 参与度（散点图）
3. 团队健康热力图（按部门）

请将图表放在"可视化图表"工作表中。
```

---

### 第三步：生成经理个人报告（可选）

基于分析结果，为每位经理生成个性化报告：

```
请根据 hr_manager_analysis.xlsx 的分析结果，
为每位经理生成个性化报告。

每个报告包含：
1. 关键指标摘要（参与度、离职率、培训完成率）
2. 优势领域（Top 3）
3. 改进机会（Top 3）
4. 具体建议（3-5 条）

为每位经理保存独立文档：
manager_report_[姓名].docx
```

---

<details>
<summary>📋 点击展开优化版提示词</summary>

```

生成一份详细的 HR 综合报告。

**报告结构**：

# HR 经理绩效分析报告

## 1. 执行摘要
- 整体参与度情况
- 离职率趋势
- 培训完成率
- 关键发现（3-5 个要点）

## 2. 数据分析
- 参与度分布（按部门）
- 离职率分析（高风险团队）
- 培训完成率与绩效相关性

## 3. 问题识别
- 高风险经理名单（参与度<70% 且离职率>15%）
- 需要关注的经理（单项指标不达标）
- 表现优秀的经理（Top 5）

## 4. 改进建议
- 短期行动（1 个月内）
- 中期行动（1-3 个月）
- 长期行动（3-6 个月）

## 5. 下一步行动
- 负责人和截止日期
- 资源需求
- 成功指标

保存为：hr_summary_report.docx

格式要求：
- 标题字体：微软雅黑，16pt，加粗
- 正文字体：微软雅黑，10.5pt
- 英文字体：Arial
```

</details>

## ❓ 常见问题

### Q1: Excel 数据读取失败怎么办？

**原因**：文件路径错误或文件被占用。

**解决**：
1. 确认文件路径正确（如 `materials/manager_metrics.xlsx`）
2. 关闭已打开的 Excel 文件
3. 告诉 OpenCode："请重新读取文件，路径是 materials/manager_metrics.xlsx"

**预防**：在提示词中明确文件路径。

---

### Q2: Word 文档字体显示为方框怎么办？

**原因**：未明确指定中文字体。

**解决**：在提示词中添加字体要求：

```
格式要求：
- 标题字体：微软雅黑，16pt，加粗
- 正文字体：微软雅黑，10.5pt
- 英文字体：Arial
```

---

### Q3: PPT 中文显示为方框怎么办？

**原因**：python-pptx 默认字体不支持中文。

**解决**：在提示词中添加字体配置：

```
使用 python-pptx 库生成 PPT：
- 页面比例：16:9
- 中文字体：微软雅黑
- 英文字体：Arial
```

---

### Q4: HR 政策问答不准确怎么办？

**原因**：政策文档复杂，AI 可能误解。

**解决**：
1. **提供更详细的上下文**：如"请只使用 benefits_policy.docx 回答"
2. **指定章节**：如"见第 3 节：401(k) 计划"
3. **迭代优化**：如"请重新回答，引用具体章节"

---

### Q5: 邮件语气不合适？

**原因**：AI 生成的语气可能不符合你的沟通风格。

**解决**：
1. **指定语气**：如"语气要正式/友好/支持性"
2. **提供示例**：如"参考以下邮件风格：[示例邮件]"
3. **多版本对比**：生成多个版本，选择最合适的
