## 2. 基础课程适配分析

### ✅ 适合基础课程的实验（5 个）

#### 实验一：法规法律指南创建

**原版内容**：
- 研究 CCPA/CPRA 法规
- 生成法律摘要（给法务团队）
- 生成法律简报（给业务部门）
- 创建对比表格

**适配方案**：
```
实验名称：法规法律指南创建
输入：法规名称（如 GDPR/CCPA/网络安全法）
输出：legal_guidance.md

核心技能：
- 网络搜索法规信息
- 生成法律摘要
- 生成业务简报
- 创建对比表格

提示词示例：
"请搜索并研究《网络安全法》，
生成法律指南。

包含：
1. 法律摘要（给法务团队）
2. 业务简报（给业务部门）
3. 关键条款对比表
4. 合规行动清单

保存为：legal_guidance.md"
```

**技术难度**：⭐⭐⭐（网络搜索 + 法律术语）  
**工作量**：4-6 小时

---

#### 实验二：合同摘要生成

**原版内容**：
- 读取供应商合同
- 生成合同摘要
- 识别关键条款

**适配方案**：
```
实验名称：合同摘要生成
输入文件：vendor_contract.docx
输出文件：contract_summary.docx

核心技能：
- 读取合同文档
- 生成合同摘要
- 识别关键条款（价格/交付/违约）

提示词示例：
"请读取 materials/vendor_contract.docx，
生成合同摘要。

包含：
1. 合同基本信息
2. 关键条款摘要
   - 价格条款
   - 交付条款
   - 违约责任
3. 风险提示
4. 建议审查点

保存为：contract_summary.docx

格式要求：
- 标题字体：微软雅黑，16pt，加粗
- 正文字体：微软雅黑，10.5pt
- 英文字体：Arial"
```

**技术难度**：⭐⭐（python-docx）  
**工作量**：4-6 小时

## 3. 高阶课程适配分析（Agent 方向）

### ✅ 适合高阶课程的实验（1 个）

#### Agent：Regulatory Inquiry Assistant（基于 Task 4-5）

**原版内容**：
- 创建监管查询助手 Agent
- 网络搜索权威法规来源
- 回答审计相关问题
- 生成有依据的答案

**Agent 设计**：
```yaml
# agent_regulatory_assistant.yaml
name: Regulatory Inquiry Assistant
description: 法规查询助手
version: 1.0

capabilities:
  - web_search: 网络搜索权威来源
  - regulatory_qa: 法规问答
  - citation: 引用来源
  - audit_support: 审计支持

knowledge_sources:
  - 政府官方网站（.gov）
  - 监管机构网站
  - 法律法规数据库

presets:
  - name: consumer_rights
    description: 消费者权利问答
    prompt: |
      请回答关于消费者权利的问题：
      - 问题：{question}
      
      要求：
      1. 只使用权威来源（.gov 网站）
      2. 每个回答必须引用来源
      3. 标注法规版本和生效日期
      4. 如有不确定性，明确说明
      
  - name: breach_notification
    description: 数据泄露通知问答
    prompt: |
      请回答关于数据泄露通知的问题：
      - 问题：{question}
      
      覆盖范围：
      1. 通知时限
      2. 通知对象
      3. 通知内容要求
      4. 权威来源引用
      
  - name: enforcement_penalties
    description: 执法处罚风险评估
    prompt: |
      请评估违规处罚风险：
      - 场景：{scenario}
      
      输出：
      1. 适用法规
      2. 处罚范围
      3. 风险矩阵（低/中/高）
      4. 纠正措施建议
      5. 权威来源引用

behavior_rules:
  - cite_sources: true
  - authoritative_only: true  # 只使用权威来源
  - flag_uncertainty: true  # 标注不确定性
  - professional_tone: true  # 专业语气

output_formats:
  - markdown: 问答结果
  - json: 结构化数据
```

**教学价值**：
- ✅ Agent 创建完整流程
- ✅ 权威来源搜索
- ✅ 引用和依据
- ✅ 审计支持场景

**工作量**：16-20 小时
