# M04 市场营销场景：使用 OpenCode 提升营销效率

> **适用对象**：市场团队、品牌专员、营销经理  
> **所需工具**：OpenCode Desktop  
> **难度**：⭐⭐⭐ 中等  
> **预计时间**：约 60-90 分钟

你不需要任何编程基础。只需要用自然语言告诉 OpenCode 你想做什么，它会自动编写 Python 代码来完成文件处理任务。

---

## 📖 智能办公工作原理

**OpenCode 如何帮你办公？**

```
你输入自然语言 → OpenCode 理解需求 → 自动编写 Python 代码 → 操作文件 → 输出结果
```

**以本模块为例**：

| 步骤 | 你做的 | OpenCode 做的 |
|------|--------|--------------|
| 1 | 输入："请合并这 3 个市场报告" | 理解需求，规划任务 |
| 2 | 等待 | 编写 Python 代码读取 Word 文档 |
| 3 | 等待 | 整合内容、补充分析 |
| 4 | 等待 | 生成统一的市场分析报告 |
| 5 | 收到文件：`市场分析报告.docx` | 汇报完成 |

**你需要知道的**：
- ✅ 不需要写代码，用自然语言描述需求即可
- ✅ OpenCode 生成的代码是透明的，你可以查看和学习
- ✅ 如果结果不满意，可以告诉 OpenCode 修改
- ✅ 生成的文件会保存在 `materials/` 目录
- ✅ **AI 的输出可能有偏差，这是正常的**——你可以像和同事交流一样不断调整

---

## 📁 课件文件

本模块使用以下示例文件（位于 `materials/` 目录）：

| 文件名 | 说明 |
|--------|------|
| `产品描述.docx` | Mystic Spice Premium Chai Tea 产品信息 |
| `市场趋势.docx` | 拉丁美洲市场趋势数据 |
| `推广计划.docx` | 产品推广计划 |
| `营销数据.xlsx` | 销售数据、社交媒体互动、搜索趋势 |

> 请将 `materials/` 文件夹放在你的工作目录下，然后打开 OpenCode Desktop，将工作目录指向该文件夹的上一级目录。

## 实验一：整合多个市场报告（Word 文档）

### 场景描述

你是 Contoso 公司拉丁美洲市场营销总监。你收到了 3 份独立的市场报告：
- 产品描述报告
- 市场趋势分析
- 推广计划

你需要将这 3 份报告整合为一份统一的市场分析报告，以便向高级管理层汇报。

---

### 第一步：整合市场报告

打开 OpenCode Desktop，在对话框中输入以下提示词：

---

#### 🟢 入门版提示词（推荐）

```
请读取 materials 目录下的以下文件：
- 产品描述.docx
- 市场趋势.docx
- 推广计划.docx

合并为一份统一的市场分析报告，包含以下章节：
1. 执行摘要
2. 产品概述
3. 市场分析
4. 推广计划
5. 竞争分析（请补充）

报告要求：
- 专业风格，适合向管理层汇报
- 使用简体中文
- 包含清晰的章节标题

保存为：LATAM 市场分析报告.docx
```

等待 OpenCode 处理完成（约 2-3 分钟）。

---

#### 🟡 优化版提示词（更详细的要求）

<details>
<summary>📋 点击展开优化版提示词</summary>

```
请读取 materials 目录下的以下文件：
- 产品描述.docx
- 市场趋势.docx
- 推广计划.docx

整合为一份完整的市场分析报告，结构如下：

**报告结构**：
1. 执行摘要（200-300 字，概述关键发现）
2. 产品概述（产品名称、特点、目标受众）
3. 市场分析（市场规模、趋势、消费者偏好）
4. 推广计划（渠道、活动、时间表）
5. 竞争分析（主要竞争对手、市场份额、差异化策略）
6. 建议与下一步行动

**格式要求**：
- 使用简体中文
- 专业商务风格
- 每个章节有清晰的标题
- 关键数据用粗体标注

保存为：LATAM 市场分析报告.docx
```

</details>

---

### 第二步：补充竞争分析（可选）

查看生成的报告后，你可能需要补充更多信息。继续输入：

```
请在 LATAM 市场分析报告.docx 中，
补充竞争分析章节的详细内容，包括：
- 主要竞争对手列表（至少 3 个）
- 各竞争对手的市场份额
- 产品对比分析
- 我们的差异化优势

请更新并保存文档。
```

---

### 预期输出

完成以上步骤后，你将得到一个 Word 文档：

| 文件名 | 内容 | 大小参考 |
|--------|------|---------|
| `LATAM 市场分析报告.docx` | 执行摘要、产品概述、市场分析、推广计划、竞争分析 | ~100-200KB |

> 💡 **注意**：如果 Word 文档已打开，OpenCode 可能无法覆盖，会生成 `_v2` 版本（见常见问题）。

## 实验二：Excel 营销数据分析

### 场景描述

你有一份过去 12 个月的营销数据，包含：
- 每月销售额
- 社交媒体互动数据
- 在线搜索趋势
- 营销活动投入

你希望用 OpenCode 帮你分析：
- 销量最高的月份
- 社交媒体互动和销量的关联
- 下个月的营销预算分配建议

---

### 第一步：分析营销数据

打开 OpenCode Desktop，在对话框中输入以下提示词：

---

#### 🟢 入门版提示词

```
请读取 materials/营销数据.xlsx，
分析以下问题：

1. 销量最高的 3 个月份
2. 社交媒体互动和销量的关联
3. 推荐下个月的营销预算分配

请将分析结果保存为新的 Excel 文件，
文件名：营销数据分析报告.xlsx
```

---

#### 🟡 优化版提示词（需要可视化时用）

<details>
<summary>📋 点击展开优化版提示词</summary>

```
请读取 materials/营销数据.xlsx，
完成以下分析任务：

**分析要求**：
1. 计算每月销售总额和增长率
2. 找出销量最高的 3 个月份并分析原因
3. 分析社交媒体互动（点赞、评论、分享）与销量的相关性
4. 分析在线搜索趋势与销量的关系
5. 推荐下个月的营销预算分配（按渠道）

**输出要求**：
- 新建"分析结果"工作表
- 包含分析摘要文字说明
- 包含数据表格
- 生成可视化图表（柱状图、折线图、散点图）

保存为：营销数据分析报告.xlsx
```

</details>

---

### 第二步：生成可视化图表（可选）

如果需要图表来直观展示分析结果，继续输入：

```
请在营销数据分析报告.xlsx 中，
生成以下图表：

1. 月度销售趋势图（折线图）
2. 社交媒体互动 vs 销量（散点图）
3. 各渠道预算分配（饼图）

请将图表放在"可视化图表"工作表中。
```

---

### 第三步：生成营销建议

基于分析结果，继续输入：

```
请根据分析结果，
生成一份营销建议文档，包含：

1. 表现最好的 3 个月份及原因分析
2. 社交媒体和销量的关联强度
3. 下个月各渠道预算分配建议（百分比）
4. 风险提示和应对措施

保存为：营销建议.docx
```

---

### 预期输出

完成以上步骤后：

| 输出 | 内容 |
|------|------|
| `营销数据分析报告.xlsx` | 原始数据、分析结果、可视化图表 |
| `营销建议.docx` | 营销建议文档 |

## 实验三：AI 头脑风暴生成营销创意

### 场景描述

你需要为 Mystic Spice Premium Chai Tea 新产品线设计营销创意方案。

- 适合拉丁美洲市场
- 针对 25-40 岁注重健康的消费者

---

### 第一步：生成营销创意

---

#### 🟢 入门版提示词

```

产品信息：
- 天然香料制成的健康饮品
- 目标受众：25-40 岁，注重健康和体验的消费者
- 品牌定位：温暖、舒适、健康、真实

要求：
- 5 个创意方案
- 每个包含：主题、标语、活动形式
- 适合拉丁美洲市场

保存为：营销创意方案.docx
```

---

#### 🟡 优化版提示词（更详细的创意要求）

<details>
<summary>📋 点击展开优化版提示词</summary>

```
**品牌信息**：
- 产品名称：Mystic Spice Premium Chai Tea
- 产品特点：天然香料、健康、温暖舒适
- 目标受众：25-40 岁，注重健康和体验的消费者

1. 创意主题（吸引人的标题）
```

### 场景描述

你有一份初稿营销文案，需要：
- 优化文案让它更吸引人

```
- 生成 3 个版本（温暖风格、专业风格、年轻风格）

- 适合社交媒体发布

保存为：营销文案合集.docx
```

---

#### 🟡 优化版提示词（更详细的文案要求）

<details>
<summary>📋 点击展开优化版提示词</summary>

## ❓ 常见问题

### Q1: 文件合并后格式混乱怎么办？

**原因**：不同文档的样式不统一。

**解决**：
1. 告诉 OpenCode："请统一文档格式，使用一致的标题样式和字体"
2. 或者手动在 Word 中使用"格式刷"统一格式

**预防**：在提示词中明确"使用统一的格式和样式"。

---

### Q2: Excel 图表中文显示为方框怎么办？

**原因**：matplotlib 默认字体不支持中文。

**解决**：输入以下提示词让 OpenCode 修复：

```
请在生成图表时，添加中文字体配置：
plt.rcParams['font.sans-serif'] = ['Microsoft YaHei', 'SimHei']
plt.rcParams['axes.unicode_minus'] = False
```

> 💡 **已优化**：本课程的提示词已包含中文字体配置要求。

---

### Q3: AI 生成的创意不够好怎么办？

**原因**：提示词不够具体，或者 AI 理解有偏差。

**解决**：
1. **补充背景信息**：提供更多产品细节、目标受众信息
2. **指定风格**：如"参考 Nike 的营销风格"或"类似 Apple 的简洁风格"
3. **迭代优化**：如"第 2 个创意不错，请再扩展 3 个类似方案"

**示例**：
```
请重新生成营销创意，参考以下要求：
- 参考 Red Bull 的极限营销风格
- 加入社交媒体挑战元素
- 预算控制在中等水平
```

---

### Q4: 文案优化后失去原有风格？

**原因**：AI 过度优化，改变了原有语气。

**解决**：
1. **指定保留元素**：如"保留原有的温暖语气"
2. **提供示例**：如"参考以下文案风格：[示例文案]"
3. **多版本对比**：生成多个版本，选择最合适的

**示例**：
```
请优化文案，但保留原有的温暖语气。
不要过于商业化，保持亲切自然的感觉。
```

---

### Q5: 如何生成特定语言的文案？

**原因**：需要针对多语言市场生成文案。

**解决**：

```
请生成西班牙语版本的营销文案，
目标市场：墨西哥、阿根廷、哥伦比亚。
保持原有的温暖风格，注意当地文化差异。
```

**预防**：在提示词开头明确"使用西班牙语"或"使用葡萄牙语"。
